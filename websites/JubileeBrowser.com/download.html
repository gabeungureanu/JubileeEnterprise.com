Installation Instructions

Step 1: Download

Click one of the download buttons above to download the installer to your computer.

Step 2: Run the Installer

Double-click the downloaded file to start the installation. If Windows SmartScreen appears, click "More info" and then "Run anyway". The installer is digitally signed and safe.

Step 3: Follow the Wizard

Follow the on-screen instructions. Accept the license agreement and choose your installation preferences. No administrator rights required for personal installation.

Step 4: Start Exploring with Jubilee

Once installation completes, Jubilee Inspire will launch automatically and greet you. You can switch between Internet mode and Worldwide Bible Web mode using the mode toggle in the toolbar.
