Version 8.0.0 Major Release
December 10, 2024

Major release introducing the dual-mode browsing experience with WWW and WWBW modes, complete browser redesign, and enterprise-ready features.

New Features

  • Dual-mode browsing: WWW MODE (Internet) and WWBW MODE (Worldwide Bible Web)
  • Visual mode indicator with yellow navigation bar for WWBW mode
  • Tab management with drag-and-drop support and mode-specific styling
  • Native inspire:// protocol handler for Scripture references
  • Automatic updates with differential downloads
  • Built-in content filtering with comprehensive blocklist
  • Session restoration across updates and restarts
  • Bookmark and history management per browsing mode
  • Settings page with customizable options

Enterprise Features

  • Silent installation with command-line options (/S flag)
  • Per-user and per-machine installation modes
  • Group Policy compatible configuration
  • Pre-configuration with JSON settings files
  • Detailed installation logging for IT administrators
  • Centralized update management options

Security

  • Process isolation between main and renderer processes
  • Session partitioning between browsing modes
  • Content Security Policy enforcement
  • Automatic update verification with code signing
  • Secure token storage for authentication

Technical

  • Built on Electron 28.3.3 with Chromium core
  • TypeScript codebase with strict type checking
  • Modular architecture with separated concerns
  • Cross-platform foundation (Windows release, macOS/Linux planned)
