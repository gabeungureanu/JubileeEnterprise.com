Common Issues

Quick solutions to frequently encountered problems.

Windows SmartScreen is blocking the installer

This is normal for new software. Click "More info" and then "Run anyway" to proceed with installation. The installer is digitally signed by Jubilee Software, Inc. and is completely safe.

How do I switch between Internet and Jubilee Bibles mode?

Use the mode toggle button in the toolbar to switch between modes. Each mode has separate cookies, history, and storage for your privacy. Jubilee Bibles mode provides access to Scripture and faith-based resources through the inspire:// protocol.

Updates are not being detected

Jubilee Browser checks for updates automatically every 4 hours. You can manually check by opening the About dialog (Settings > About). Make sure your firewall allows connections to updates.jubileebrowser.com.

The browser won't start

Try running the browser from the command line to see any error messages. If the issue persists, try renaming the user data folder at %LOCALAPPDATA%\JubileeBrowser to reset settings.

How do I deploy to my church or school?

Jubilee Browser supports silent installation for easy deployment. Use the command JubileeSetup.exe /S /ALLUSERS for silent installation. See our Churches & Schools page for detailed deployment instructions.

Can I customize the content filtering?

Jubilee Browser's content filtering is built into the browser to protect all users by design. Organizations can deploy additional custom blacklists for their specific needs. Contact us for help with organizational deployment.
