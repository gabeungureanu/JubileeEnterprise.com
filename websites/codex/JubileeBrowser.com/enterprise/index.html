The Challenge You Face

Church IT teams and Christian schools often struggle to manage device policies, filtering systems, and browser configurations. Keeping students and staff safe online requires constant attention, multiple tools, and ongoing maintenance that takes time away from your mission.

Complex Filtering Tools

Most web filters require separate software, subscriptions, and constant updates. They can be bypassed and need regular maintenance.

Multiple Devices, Multiple Problems

Managing browser settings across church offices, classrooms, labs, and family computers creates a fragmented, difficult-to-maintain environment.

Limited IT Resources

Most churches and schools don't have dedicated IT staff. Technology decisions fall on volunteers or staff with other responsibilities.

Deployment Guide

Command-Line Options

Silent Installation (Per-User)

JubileeSetup.exe /S

Installs for the current user only. No admin rights required.

Silent Installation (All Users)

JubileeSetup.exe /S /ALLUSERS

Installs for all users on the machine. Requires administrator privileges.

Silent Installation with Logging

JubileeSetup.exe /S /LOG=C:\Logs\install.log

Writes detailed installation log for troubleshooting deployment issues.

Pre-Configuration

Set defaults and lock settings before deployment.

Default Settings (settings.json)

{
  "homepage": "inspire://home",
  "searchEngine": "duckduckgo",
  "defaultMode": "jubilee",
  "autoUpdate": true
}

Place in the resources folder to set defaults for new installations.

Locked Policies (policies.json)

{
  "homepage": {
    "value": "inspire://home",
    "locked": true
  },
  "autoUpdate": {
    "value": true,
    "locked": true
  }
}

Lock settings to prevent users from changing them.
